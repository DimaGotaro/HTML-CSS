Введение

Чтобы создавать веб-сайты, вы должны знать о HTML (HyperText Markup Language / Язык гипертекстовой разметки) — фундаментальной технологии, которая используется для определения структуры веб-страницы. HTML применяется для того, чтобы определить как должен отображаться ваш контент в различном виде:

  1. Абзац
  2. Список
  3. Заголовок
  4. Ссылка

Существует множество и других доступных элементов, также вы можете создать свой собственный новый элемент.

Темы для изучения

Для того чтобы создать свою первую страницу с помощью HTML нужно понимать некоторые темы:

  • Работа с текстом
  • Атрибуты
  • Ссылки
